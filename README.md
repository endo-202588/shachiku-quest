## アプリタイトル
社畜クエスト　〜デスクの上の大冒険〜

## サービス概要
**社畜クエスト**は、チーム内の業務量を可視化し、助け合いを促進する業務管理アプリです。

### 解決したい課題
- 誰がどのくらい忙しいのか分からない
- 助けたいけど、誰が助けを求めているか分からない
- 業務の偏りが見えにくい
- 管理職が社員の状況を把握しづらい

### 提供する価値
- その日の業務量を「ステータスコード」で簡単に共有
- 助けられる人と助けが必要な人をマッチング
- RPGゲームの世界観で楽しく協力を促進
- 管理職は社員の状況を一目で把握できる

---

## このサービスへの思い・作りたい理由

普段誰がどのくらいの業務負担があるのか、なかなか把握できないことをもどかしく思っていました。

そこで、"猫の手も借りたい"人と、"助けられます"という人を楽しくつなげて業務量の平均化を図り、助けた人も経験値を積むことでレベルアップしていく様子を楽しめるアプリを作りたいと考えました。

実用的な面では、その日のチームメイトの業務量が可視化され、お休みも含めて社員の状況が把握できる、管理職にとっては簡単に社員の負担度がつかめる、というメリットがあります。

## サービスの利用イメージ

### 1. 朝の業務開始時
各社員（勇者）は、出勤したらその日の業務量を登録します。

**ステータスコード（業務量）**
- 🟢 **平和** - 余裕があります（ヘルプ不要）
- 🟡 **疲れ気味（0.5h）** - ちょっと忙しい
- 🟠 **やばいかも（1h）** - かなり忙しい
- 🔴 **きてます（1.5h）** - 助けてください！
- ⚫ **昇天（2h〜）** - 限界です！

### 2. シャチークの酒場（ヘルプマッチング）
助けられる人は「使える魔法（お手伝い可能時間）」を登録します。

**使える魔法**
- ⚡ **ホイミ（0.5h）** - ちょっとだけお手伝い
- ✨ **ベホイミ（1h）** - しっかりお手伝い
- 💫 **ベホマ（1.5h）** - がっつりお手伝い
- 🌟 **ベホマズン（2h〜）** - 全力でお手伝い

### 3. ヘルプ依頼
- 助けが必要な人は「シャチークの酒場」でヘルプを依頼
- 助けられる人は依頼を受けてタスクを完了
- 完了すると「徳（ポイント）」を獲得

### 4. レベルアップ
一定の徳を得ると、レベルがアップします！

**レベル一覧**
- 🌱 **旅人** - 初心者（0〜99徳）
- 🌿 **先輩** - 経験者（100〜299徳）
- 🌳 **先生** - ベテラン（300〜599徳）
- 🏆 **達人** - エキスパート（600〜999徳）
- 🙏 **仏** - マスター（1000徳〜）

## サービスの差別化ポイント・推しポイント
業務管理アプリのようなものは多く存在していますが、このように"助けを求める、助ける"といったアプリはあまり見られません。
しかもそれをRPGゲームの代表作"ドラゴンクエスト"のパロディのような形でゲーム仕様にし、社員が楽しみながら協力することを促進するという発想は非常にユニークであり、実用的でありながらもエンタメ性のあるアプリになることが期待できると思います。

## 使用する技術スタック

### デプロイ先
